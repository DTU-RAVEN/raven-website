# Our First Project: Indoor Mapping Drone

After much deliberation, we're excited to announce that our first major project will be developing an autonomous drone specialized for indoor mapping and navigation.

## Project Goals

1. Create a lightweight drone capable of navigating indoors without GPS
2. Develop SLAM (Simultaneous Localization and Mapping) algorithms for real-time mapping
3. Implement obstacle avoidance using computer vision and depth sensing
4. Build a user interface for mission planning and map visualization

## Technical Specifications

Our drone will include:

- Custom carbon fiber frame for lightweight rigidity
- Multiple depth cameras and obstacle sensors
- Onboard computer for real-time processing
- Custom flight controller for precise indoor navigation

## Timeline

We've planned the following milestones:

| Phase | Description | Timeline |
|-------|-------------|----------|
| Research | Component selection and design decisions | May-June 2025 |
| Prototyping | Building initial hardware and software | July-August 2025 |
